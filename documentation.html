Giveaway Galore Documentation

Proposal

API

The api I am using is GamerPowers giveaway api. It allows anyone to access the best giveaway in gaming including beta keys and in-game loot. The api doc can be accessed here.

Features

The main feature of my app is to display a list of all the giveaways available through the api. There will be filter options that the user can select to narrow down the searches. The app will allow the user to choose what platform the giveaway is for, the type of giveaway it is (eg: game, beta keys, loot) and then the user can sort the results by its date, value, or popularity. The app will show all the available giveaways that match the search filters and give information on each giveaway. It will include a link to the giveaway so the user can easily enter. Lastly the user can favorite an individual giveaway as to keep track of it and it will show up in the favorites page so they can easily check up on its status. The app will also include a page that shows what giveaways other users who used the page have liked to be able to see whats popular among app users.
